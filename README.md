# Change-Life OS

一個廣東話、瀏覽器優先嘅人生管理 OS。核心閉環係：

`反願景 → 90 日主線 → 今日唯一優先 → 深度工作 → 3-2-1 復盤`

## MVP 功能

- 四步首次設定：反願景、理想願景、個人利基、90 日主線。
- HUMAN 3.0：心智、身體、精神、天職每日狀態。
- RPG 任務：主線、支線、Boss 戰、XP、Level、連續行動日數。
- 25／50／90 分鐘深度工作計時器。
- Idea → Learn → Teach → Sell 內容飛輪及 PIA 檢查。
- 3-2-1 晚間復盤、一日重置、30 日 Monk Mode 承諾。
- 瀏覽器本機儲存、JSON 匯出及安全匯入。

## 本機運行

需要 Node.js 22.13 或以上。

```bash
npm install
npm run dev
```

開啟終端顯示嘅本機網址。

## 驗證

```bash
npm test
```

測試會檢查 XP、Level、任務反轉、連續日數、PIA、備份驗證、正式建構及伺服器輸出。

## 資料與部署

所有個人資料預設只儲存喺使用者目前瀏覽器。網站以 Cloudflare 相容靜態／邊緣輸出建構，`npm run build` 會產生正式版本。
